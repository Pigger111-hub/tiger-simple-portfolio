# Portfolio Deployment Plan: Tiger Phanyawich

## Overview
Build a high-performance, dark-themed data analyst portfolio website hosted on GitHub Pages using vanilla HTML, CSS, and JS.

## Phase 1: Project Setup & Structure
- Initialize a local project folder named `portfolio`.
- Create the following file structure:
  - `index.html` (Main structure)
  - `style.css` (Dark theme, JetBrains Mono font, responsive layout)
  - `script.js` (Simple interactions)
  - `.nojekyll` (Ensure assets are correctly processed by GitHub Pages)

## Phase 2: Design & Implementation
1. **Skeleton (`index.html`):** Semantic HTML5 structure for Hero, Projects, and Contact sections.
2. **Styling (`style.css`):** 
    - Modern dark theme (e.g., `#0f172a`).
    - Integrate JetBrains Mono font.
    - Grid-based layout for 3 project cards.
    - "AI/tech" aesthetic (subtle borders, clean spacing).
3. **Content:** Input personal details and professional project summaries.

## Phase 3: Deployment
1. **Repository:** Create a repository on GitHub (e.g., `portfolio` or `your-username.github.io`).
2. **Push:** Initialize Git and push the local project to the GitHub remote repository.
3. **Configuration:** Navigate to GitHub Settings > Pages > Enable "Deploy from a branch" from the main branch.
