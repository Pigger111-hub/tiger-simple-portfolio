# Tiger Phanyawich | Data Analyst Portfolio

Welcome to my professional portfolio! This repository contains my data analysis projects, demonstrating my skills in SQL, Python, Tableau, and Power BI.

## Portfolio Structure
- `docs/`: Contains the live website files.
  - `index.html`: Main portfolio structure.
  - `style.css`: Modern dark/light theme styling.
  - `script.js`: Theme switching functionality.
- `plan/`: Contains the development plan for this project.

## Tech Stack
- **Languages:** HTML5, CSS3, Vanilla JavaScript.
- **Styling:** Custom dark/light theme, JetBrains Mono font.
- **Deployment:** Hosted on GitHub Pages.
